# 🛺 Uber Auto — Ride Operations Dashboard

A comprehensive analytics dashboard built to analyze Uber Auto ride operations across key performance dimensions including bookings, revenue, distance, ratings, and zone-level demand patterns.

![Dashboard Preview](assets/dashboard-preview.png)

---

## 📊 Overview

This dashboard provides actionable insights into Uber Auto's operational data, enabling data-driven decisions around driver deployment, pricing strategy, and customer satisfaction.

### Key Metrics
| Metric | Value |
|---|---|
| ✅ Completed Bookings | 93,000 |
| ❌ Lost Bookings | 57,000 |
| 💰 Total Revenue | ₹51.85M |
| 📍 Total Distance | 2.51M km |
| 🛣️ Avg Ride Distance | 24.64 km |
| ⭐ Customer Rating | 4.40 / 5 |
| 🚗 Driver Rating | 4.23 / 5 |

---

## 🔍 Key Insights

- **Top Pickup Zone:** Khandsa (600 rides) — high-demand origin point for driver positioning
- **Top Drop Zone:** Ashram (592 rides) — key destination for return-trip planning
- **Revenue Leader:** Auto-rickshaws generate ₹13M, outperforming all other vehicle types
- **Seasonal Dip:** Bookings drop sharply in Feb–Mar, signaling an opportunity for dynamic pricing
- **Lost Booking Gap:** 57K lost vs 93K completed — a 38% opportunity gap worth investigating
